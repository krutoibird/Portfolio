Projects

FocusBot

Problem: I kept getting distracted by my phone while working and wanted something that would actually catch me in the act, not just a screen-time number at the end of the day.

Action: I built a desk device on a Raspberry Pi 5 that uses YOLOv8 to detect phone use in real time. If a phone is visible for more than 5 seconds, it triggers a visual alert and plays an AI-generated voice reminder through the Groq API.

Result: A working device that detects phone use live and tracks distraction time and alert count per session, giving me real feedback on my focus habits.

Python · YOLOv8 · Groq API · Raspberry Pi · TTS

Challenges
  • The camera cable kept corrupting the SD card, which meant reinstalling everything from scratch multiple times
  • Had to manually configure the speaker to get audio working
  • A small code mistake caused the program to crash on every frame with no detections
  • The AI model I was using got shut down mid-project and I had to switch to a different one
  • The Pi kept getting blocked when making API requests - fixed it by changing how the request identified itself
  • Ran out of storage on the Pi mid-install and had to find a workaround

SWIFT Program Website

Problem: Prospective Grade 11/12 students had no central place to learn about the SWIFT program — the main source was a physical poster.

Action: My group partner and I built a 6-page website covering admissions, field trips, projects, and staff, with a reusable JS navbar and an embedded live Google Calendar.

Result: A complete informational site that gives prospective students one clear place to understand the program.

HTML · CSS · JavaScript · Google Calendar API

Behind the Scenes
  • Information about the program was limited — the main source was a physical poster, leaving some pages sparse
  • All layout and styling was done by hand in CSS with no framework — spacing, alignment, and consistency across 6 pages required a lot of trial and error
  • Built a reusable navbar component in JavaScript that loads dynamically across all pages
